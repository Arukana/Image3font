the image3font script¶

Image3font is a image's converter to multi-colours font written in FontForge/FontTools's Hy language and the futur of Image2font's repository.

usage¶

see command help:

Usage:  [options] [IMAGES]...

  Options:
    -h, --help            show this help message and exit
    -v, --version         show program's version number and exit
    -m MANIFEST, --manifest MANIFEST specify the manifest.
